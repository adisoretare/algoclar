## Ideea de maxim și minim curent

Menținem un **maxim curent** și îl actualizăm la fiecare cifră extrasă:

```
Pornim: maxim = -1
Cifra 3: max(-1, 3) = 3  → maxim = 3
Cifra 9: max(3, 9)  = 9  → maxim = 9
Cifra 2: max(9, 2)  = 9  → maxim = 9
Cifra 7: max(9, 7)  = 9  → maxim = 9
Rezultat: 9
```

<ObservationBox>
Inițializează `maxim = -1` și `minim = 10`. Astfel orice cifră (0–9) va actualiza corect valorile la prima comparație. Dacă inițializezi `minim = 0`, nu vei detecta că minimul este 0 când există cifra 0.
</ObservationBox>

## Implementare C++

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    if (n == 0) {
        cout << "Max: 0, Min: 0" << endl;
        return 0;
    }

    int maxim = -1, minim = 10;
    int temp = n;

    while (temp > 0) {
        int cifra = temp % 10;
        if (cifra > maxim) maxim = cifra;
        if (cifra < minim) minim = cifra;
        temp /= 10;
    }

    cout << "Max: " << maxim << ", Min: " << minim << endl;
    return 0;
}
```

**Exemplu:** Input `3927` → `Max: 9, Min: 2`

<MistakeBox>
**Greșeala 1:** Inițializezi `maxim = 0`. Dacă toate cifrele sunt 0 (cazul n = 0), `maxim` rămâne 0 — aparent corect, dar nu ai parcurs nicio cifră. Inițializarea cu `-1` e mai sigură.

**Greșeala 2:** Inițializezi `minim = 0`. Pentru numărul 123, `minim` rămâne 0 deși 1 e mai mic — pentru că 0 < 1. Inițializează `minim = 10` (mai mare decât orice cifră).

**Greșeala 3:** Compari cu `temp` (valoarea în schimbare) în loc de `cifra = temp % 10`. Extrage cifra mai întâi, compară cu ea.
</MistakeBox>
